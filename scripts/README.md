# Scripts

This directory contains developer and maintenance scripts for the zWork project.

## Available scripts

| Script | Purpose |
|--------|---------|
| `add_docstrings.py` | Adds docstrings to undocumented functions across sidecar Python modules and commits each file separately |

## Usage

Run scripts from the repository root with Python 3.12+:

```bash
python3 scripts/add_docstrings.py
```

All scripts assume they are invoked from the repository root directory.

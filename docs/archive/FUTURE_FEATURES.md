# Future Features (Brainstorm)

## UI Control (Accessibility/RPA)
- Find UI elements, click, type, scroll, drag/drop, keyboard shortcuts
- Window management (move/resize/snap), app launching, focus switching
- Element targeting via labels/roles, fallback “user confirm target” mode

## Browser Automation
- Tab/session awareness, page/DOM/text extraction
- Form filling with “stop-before-submit”
- Download handling (monitor, rename, move), profile isolation

## Screen Understanding
- Screenshots and region selection
- OCR and text/structure extraction from pixels
- Element anchoring (“click the blue button”), visual change detection

## File System + Storage
- Advanced search (name + content), indexing, tagging
- Safe bulk operations with previews/diffs
- Undo/snapshots, archive/restore

## Document Understanding
- PDF/Office parsing and structured extraction (tables/forms)
- Redaction, templating, merge/fill workflows
- Export pipelines (Markdown/PDF/email)

## Email / Messaging
- Draft/rewrite, thread summarization
- Send with approvals, address book/contact lookup
- Attachment management and filing

## Calendar / Tasks
- Schedule proposals, conflict detection
- Meeting prep packs, follow-up task creation
- Reminders, timeboxing

## Clipboard + Snippets
- Clipboard history
- Structured paste (table/CSV/Markdown)
- Snippet library and transformations (copy → format → paste)

## App Connectors (Local-first)
- Notes/Notion/Obsidian
- Slack/Teams
- Jira/Linear
- Google Workspace/Microsoft 365
- CRM tools (where applicable)

## Knowledge Base
- Local index (notes, docs, repos) + semantic search
- Citations and “source-of-truth” pinning
- Per-project context profiles

## Workflow Builder
- Record/replay
- Parameterized variables and prompts
- Branching/conditions, retries
- Scheduling/triggers, templates/runbooks

## Safety & Governance
- Permission scopes (folder/site/app/time)
- Risk classification + confirmations
- Dry-run mode
- Action logs/audit trails
- Secret detection and policy rules

## Observability / Debugging
- Step traces
- Screenshots/DOM snapshots during actions
- Reproducible run reports
- “Why did you do that?” explanations

## Network Tools
- HTTP client, webhook testing
- OpenAPI import
- OAuth helpers
- Request replay

## Data Tools
- CSV/Excel read/write
- SQL (local/remote)
- ETL transforms, dedupe/merge
- Validation rules, chart/report generation

## Media Tools
- Audio transcription
- Meeting recording summarization
- Basic image editing (crop/annotate)
- Video timestamp notes

## System Tools
- Process management
- Environment inspection
- Package management helpers
- Git operations
- Backup/restore
- Notifications

## Collaboration
- Shareable workflows/runbooks
- Handoff bundles (artifact + log + sources)
- Team-safe redaction
- Exportable audit trails
